Who I Am

I'm a WordPress developer obsessed with the intersection of clean code and thoughtful design. I've been building on WordPress for over 3.5 years — custom themes, complex plugins, WooCommerce stores, and headless setups using the REST API & GraphQL.

My philosophy: WordPress should be fast, maintainable, and a joy to use — not just for developers, but for the clients managing it daily.

When I'm not deep in PHP or block editor internals, I'm exploring performance optimisation, accessibility standards, and modern JS tooling.
